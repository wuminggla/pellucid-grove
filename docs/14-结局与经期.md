# 14 · 结局系统 + 经期系统

> 两个纯机制系统(不依赖内容草稿)。模块:`src/game/endings/` `src/game/cycle/`。

## 结局系统（endings）
> 落地 阶段指导与结局_v1 + 设计正典 §10。本模块管"通关型"结局;软卡死/硬失败由别处管。
- **三结局**(状态驱动):解脱(金盆洗手)/畸形团体/堕落生育。
- `endingTendency(state)` → 过程可感知的 _结局倾向(状态栏实时显示玩家在滑向哪):死撑/动摇=解脱;崩溃=畸形团体;母猪化/怀孕=堕落生育。
- `resolveEnding(state, {revengeComplete})`:怀孕→提前锁堕落生育(可在复仇前);否则需复仇完成→按倾向出结局;未达→null(进行中)。
- `isSalvationOpen(state)`:金盆洗手可达性=堕落<50 且 认知未进崩溃(用户定不过50)。`SALVATION_CORRUPTION_CAP=50`。
- 复仇完成判定:`turf.isRevengeComplete(regions)`=最终boss区域(city)已击败。
- EngineState 加 `pregnant?`(生育线判定后置占位)。
- **软卡死**(连续请假)=请假机制+滑动窗口保底;**硬失败**(极道威望连2次0)=settleDaily.hardFail。两者已在别处实现。
- 待后续:避孕套"废除"态细化畸形团体条件;学籍(乙接口)纳入解脱门槛;结局演出UI;_结局倾向接状态栏。

## 经期系统（cycle）
> 设计正典批注:去医学化,只分安全期/危险期。
- `CYCLE`(占位:14天周期,第6-9天危险期)。`isDangerDay`/`advanceCycle`(每日推进+翻转 isDangerousPeriod)。
- settleDaily 每日推进经期 → 翻转 `isDangerousPeriod`。
- **危险期效果**:套消耗×1.5(condomCost 已按 isDangerousPeriod 自动)、受孕率↑(`DANGER_CONCEPTION_MULT=3` 供生育线)、在场人数易刷高(prompt场景行已显示安全/危险期)。
- EngineState 加 `cycleDay?`。
- 待后续:受孕判定接生育线;周期数值调平。
