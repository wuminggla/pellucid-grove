# 08 · 统一事件模型（events）

> 模块：`src/game/events/`。落地 v3 蓝图 §0 最高统一模型。
> 核心：扩张长出选项，堕落让选项从SFW翻面成NSFW，翻面的"第一次"=特殊事件。所有事件归一，无特例。

## 数据模型（types.ts）
- **EventOption**：一个事件选项 = id/label/period/shape + SFW范式 + NSFW范式 + 侵蚀闸门 + 首次里程碑 + ♥/置顶/供奉标记。
- **EventShape**：`dual`双面型(有SFW+NSFW) / `born_nsfw`天生NSFW(供奉/AV) / `born_sfw`天生SFW(纯经营)。
- **ErosionGate**：侵蚀闸门，可组合多数值条件(corruption/cognition/infamy/thugs/custom)，全满足才翻面。占位堕落度≥50，留多数值接口。
- **FirstMilestone**：首次里程碑特殊事件(ledgerKey账本键/范式/堕落权重/优先级)。
- **ParadigmRef**：范式引用(worldbookKey指向世界书；inlinePrompt支持AV动态范式运行时拼装)。

## 核心逻辑（machine.ts）
- **resolveEvent(opt, ctx, fastForward)**：解析当前该怎么演 →
  - dual：闸门未开=SFW(ai_brief) / 开+首次未触发=首次特殊(ai_full,加堕落) / 开+首次已触发=NSFW常规(ai_normal)
  - born_nsfw：永远NSFW；first未触发=首次特殊
  - born_sfw：永远SFW
- **renderMode四档**：ai_full(首次,压过快进) / ai_normal(NSFW非首次正常生成) / ai_brief(SFW略写) / fast_summary(快进)
- **gateOpen**：侵蚀闸门多条件判定
- **buildMenu**：列时段菜单，NSFW态加（♥），置顶项排最前，时段过滤，未解锁不出现
- **scanForced**：强制事件候选池(优先级+已触发标签)，覆盖避孕套三连(同条件依次触发)
- **markMilestone**：消费首次后打账本标签(纯函数)
